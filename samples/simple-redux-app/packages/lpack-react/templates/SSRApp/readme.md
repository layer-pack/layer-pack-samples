## How to start

### using docker-compose

```
docker-compose up
```

& start browser at localhost:8080

### using local setup

#### setup
```
npm i
```

#### dev
```
npm run start-dev
npm run runBrowser
```

#### prod
```
npm run build
npm run start
```
